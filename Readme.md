# 📝 Notes App

A simple Notes App built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add Notes
- Delete Notes
- Notes saved using Local Storage
- Notes remain after refresh
- Responsive UI
- Input validation

## 🛠 Tech Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

```bash
notes-app/
│── index.html
│── style.css
│── script.js
```

- Event Listeners
- Dynamic Element Creation
- Local Storage
- Arrays & State Management
- Responsive UI Basics
